# TaskManager с логированием и трассировкой

Проект реализует базовые операции менеджера задач:
- добавление задачи (`Add`);
- удаление задачи (`Remove`);
- вывод списка задач (`List`);
- ведение логов в консоль и файл `task_manager.log`.

Для каждой операции добавлена трассировка:
- начало операции (`TRACE START`);
- конец операции (`TRACE END`);
- длительность (`duration_ms`);
- результат (`SUCCESS`, `WARNING`, `ERROR`).

## Запуск

```bash
python main.py
```

## Структура

- `task_manager.py` — класс `TaskManager` и контекст трассировки;
- `main.py` — демонстрационный запуск операций;
- `task_manager.log` — логи и трассы выполнения;
- `REPORT.md` — отчет по выполненной работе.
